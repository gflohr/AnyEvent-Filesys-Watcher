=head1 NAME

AnyEvent::Filesys::Watch::Backend::KQueue - BSD file system watcher

=head1 SYNOPSIS

	die "AnyEvent::Filesys::Watch::Backend::KQueue should not be used"
	    . " directly. Use AnyEvent::Filesys::Watch instead";

=head1 DESCRIPTION

This is the Linux backend for L<AnyEvent::Filesys::Watch>.  It uses
L<IO::KQueue> internally.

=head1 CONSTRUCTORS

The following constructors are available:

=over 4

=item B<new(WATCH)>

Returns the watcher. B<WATCH> should be the calling
L<AnyEvent::Filesys::Watch>.

=back

=head1 SEE ALSO

L<IO::KQueue>, L<AnyEvent::Filesys::Watch>, perl(1)
