=head1 NAME

AnyEvent::Filesys::Watch::Backend::Fallback - Fallback file system watcher

=head1 SYNOPSIS

	die "AnyEvent::Filesys::Watch::Backend::Fallback should not be used"
	    . " directly. Use AnyEvent::Filesys::Watch instead";

=head1 DESCRIPTION

This is the fallback backend for L<AnyEvent::Filesys::Watch>.  It checks
the file system in short intervals.  This is, of course, quite inefficient.

=head1 CONSTRUCTORS

The following constructors are available:

=over 4

=item B<new(WATCH)>

Returns the watcher backend. B<WATCH> should be the calling
L<AnyEvent::Filesys::Watch>.

=back

=head1 SEE ALSO

L<AnyEvent::Filesys::Watch>, perl(1)
